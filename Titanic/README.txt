В этом задании требовалось построить прогнозирующую модель, которая ответит на вопрос: 
«Какие люди имеют больше шансов выжить?» с использованием данных о пассажирах (т. е. имени, возраста, пола, социально-экономического класса и т. д.).

Я построил так называемую модель случайного леса. Эта модель состоит из нескольких «деревьев»,
которые будут индивидуально учитывать данные каждого пассажира и голосовать за то, выжил ли он. 
Затем модель случайного леса принимает демократическое решение: побеждает результат, набравший наибольшее количество голосов!
