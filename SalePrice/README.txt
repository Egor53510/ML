В этом задании, состоящем из 79 объясняющих переменных, описывающих (почти) каждый аспект жилых домов в Эймсе, штат Айова, вам предстоит предсказать окончательную цену каждого дома.
Для решения этой задачи я использовал RandomForestRegressor из библиотеки sklearn. Случайный лес использует множество деревьев и делает прогноз путем усреднения прогнозов каждого компонента дерева.
